### Ecosystem Project:
- [Mayar CLI](https://github.com/mayarid/mayar-cli "Mayar CLI") - Command line interface for Mayar API, designed for agents
- [Mayar Balance Chrome Ext](https://github.com/moerdowo/mayar-chrome-ext "Mayar Balance Chrome Ext") - Check Mayar Balance, Transactions and Product quickly
- [Mayar Quick Tools](https://github.com/moerdowo/mayar-quick-tools "Mayar Quick Tools")  - Create QRIS, Pay Request, Invoice quickly

### Fun Project:
